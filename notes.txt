simplifying if statements:
        int income = 200_000;
        boolean hasHighIncome = false;
        if (income > 100_000)
            hasHighIncome = true;

instead write like:
        int income = 200_000;
       boolean hasHighIncome = (income > 100_000)
       /this returns a true or false - no need for the if statement

ternary operator:
        int income = 200_000;
        String className = income > 100_000 ? "First" : "Economy";
